# Music Bot

A high-quality, feature-rich Discord music bot built with **discord.js** and **Lavalink**. Designed for performance, stability, and a premium user experience.

## Documentation

Full docs:

https://afteryparty.github.io/afterparty-groovy-music/

## Features

### 🎵 Core Music Playback
- **High Quality Audio**: Powered by Lavalink for lag-free streaming.
- **Multi-Platform Support**: Plays from YouTube, SoundCloud, Twitch, and more.
- **Queue Management**: Add, remove, shuffle, and loop tracks easily.
- **Rich "Now Playing" Interface**: Interactive buttons for playback control.

### ✨ Groovy+ Premium Features
Unlock the full potential of your music experience with **Groovy+**:
- **Interactive Controls**: Advanced control via Dropdown Menus (Volume, Seek, Filters, EQ).
- **Multi-Platform Access**: Exclusive access to **Spotify** and **Apple Music** playback.
- **Expanded Playlists**: Create and manage playlists with up to **200 tracks** (vs 30 for free users).
- **Dedicated Setup Channel**: A persistent, auto-updating request channel for easy song requests.
- **Audio Filters**: BassBoost, Nightcore, Vaporwave, and more.
- **24/7 Mode**: Keep the bot in the voice channel even when inactive.

## Commands

| Command | Description |
|---------|-------------|
| `/play` | Play a song from a URL or search query |
| `/stop` | Stop playback and clear the queue |
| `/skip` | Skip the current track |
| `/volume` | Adjust the playback volume |
| `/queue` | View the current song queue |
| `/filter` | Apply audio filters (BassBoost, etc.) |
| `/setup` | Create a dedicated song request channel |
| `/profile` | View your user profile and premium status |
